Font-Text-Aligns-Shadows

1.Fonts

changing styles with fonts and colors
.box1 {
        font-family: system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI",
          Roboto, Oxygen, Ubuntu, Cantarell, "Open Sans", "Helvetica Neue",
          sans-serif;
        font-size: 1.1rem;
        font-weight: bolder;
        color: #fff;
        background-color: green;
    }
.box1 pre {
        background-color: #45418d;
    }

2.Fonts and Shadow

changing styles with fonts, colors and giving shadow
.box2 {
        font-family: Arial, Helvetica, sans-serif;
        font-size: 1.1rem;
        font-weight: bold;
        color: #fff;
      }
.box2 > * {
        text-decoration: underline;
        text-shadow: 0px 0px 10px rgb(2, 110, 2);
      }
.box2 pre {
        text-decoration: none;
    }

3.Fonts,Shadow and Text-Aligns

changing styles with fonts, colors, giving shadow, aligning texts and letter-spacing 
.box3 {
        font-family: Verdana, Geneva, Tahoma, sans-serif;
        font-size: 1.1rem;
        font-weight: bold;
        color: #026a1a;
    }
.box3 > * {
        text-shadow: 0px 0px 10px rgb(54, 226, 238);
    }
.box3 h1 {
        text-align: center;
    }
.box3 h5 {
        text-align: right;
    }
.box3 pre {
        letter-spacing: 3px;
    }
